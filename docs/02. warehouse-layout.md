# SCALES - Stack-Based Layout Automizer
### 1. The Layout Model
* The warehouse is represented as a 2D Grid of Stacks. Each stack is a unique location_id.

* Property: max_stacks (How many pallets can be stacked vertically).

* State: current_occupancy (How many pallets are currently STORED here).

### 2. The Automizer (Generator)
The agent must provide an Admin interface to "Bulk Create" locations:

* Fields: Zone Prefix (e.g., "A"), Start Number, End Number, Default Max Height.

* Logic: On "Generate," create loop-based entries in warehouse_locations for the active company_id.

### 3. Visualization (The Heatmap)
* Use Konva.js or a simple CSS Grid to show the warehouse floor:

* Empty Stack: Green Border.

* Partial Stack: Yellow Fill.

* Full Stack: Red Fill (Disable "Inbound" to this ID).

* Interactive: Tapping a stack shows the "Pallet Details" (Batch, Product, Qty).

### 4. Master Settings Integration
* Zone Rules: Admins can set which products are allowed in which zones (e.g., "Zone A is for Product X only").
